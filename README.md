# CodeAlpha Python Programming Internship

Welcome to my **CodeAlpha Python Programming Internship** repository! This repository contains the projects and tasks completed as part of the Python Programming Internship offered by **CodeAlpha**. Each project demonstrates practical Python concepts and real-world problem-solving skills.

---

## 📖 About the Internship

The CodeAlpha Python Programming Internship is designed to provide hands-on experience in Python development through practical assignments. It focuses on strengthening programming fundamentals, logical thinking, file handling, automation, and object-oriented programming while encouraging the use of Git and GitHub for version control.

---

## 🎯 Internship Tasks

This repository includes the internship tasks assigned by CodeAlpha.

### ✅ Task 1 – Hangman Game

A console-based Hangman game where the player guesses a hidden word one letter at a time.

**Concepts Used**

* Random module
* Loops
* Conditional statements
* Lists
* Strings

---

### ✅ Task 2 – Stock Portfolio Tracker

A Python application that calculates the total value of a stock portfolio using predefined stock prices and user input.

**Concepts Used**

* Dictionaries
* User Input
* Arithmetic Operations
* File Handling (Optional)

---

### ✅ Task 3 – Task Automation with Python Scripts

A Python automation script to simplify repetitive tasks such as:

* Moving image files
* Extracting email addresses from text files
* Scraping webpage titles

**Concepts Used**

* `os`
* `shutil`
* `re`
* `requests`
* File Handling

---

### ✅ Task 4 – Basic Chatbot

A simple rule-based chatbot that responds to predefined user inputs.

**Concepts Used**

* Functions
* Loops
* Conditional Statements
* Input/Output
